# Algorium UK - Digital Agency Project

Professional high-end digital agency website for the UK market, built with a "Posh Engineering" aesthetic.

## 🚀 Key Features
- **Experience:** Awwwards-inspired minimalist UI/UX with a "Quiet Luxury" feel.
- **Frontend:** Next.js (App Router) + Framer Motion for cinematic animations.
- **Interactions:**
  - **Magnetic Buttons:** Tactile CTA interactions.
  - **World Clocks:** Real-time analog clocks for London, NYC, Tokyo, etc.
  - **Project Reveal:** List-based hover image reveal with premium project assets.
  - **Smart Navigation:** Hide-on-scroll sticky nav with mobile hamburger overlay.
- **Backend:** Ultra-fast Hono.js API handling contact enquiries.
- **Design System:** Structured design tokens in `pencil.md`.

## 🛠️ Project Structure
- **/frontend**: Next.js ecosystem with custom-built motion components.
- **/backend**: Hono.js/TypeScript - Lightweight API for form processing.
- **/planning**: Business strategy and technical requirements.

## 🚦 Getting Started
### 1. Frontend
```bash
cd frontend
npm install
npm run dev # Runs on http://localhost:3000
```

### 2. Backend
```bash
cd backend
npm install
npm run dev # Runs on http://localhost:3001
```

## 🎨 Design Principles
We follow the "Posh Engineering" ethos: Extreme clarity, high-contrast negative space, and a bold mix of Modern Sans (Outfit) and Elegant Serif (Italic) typography.
